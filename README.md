# Locksmith Companion Database

UK-focused automotive locksmith database used by the Auto Locksmith Companion app.

## Active workflow

The project workflow is locked in:

- [`Docs/LOCKED_DATABASE_WORKFLOW.md`](Docs/LOCKED_DATABASE_WORKFLOW.md)

The current priority is to finish Ford using complete Locksmith Cards containing practical key-stock and programming data. Photos and non-essential presentation work must not delay the first usable release.

## Current shared references

Only four shared reference categories are required for the first usable release:

- Blades
- Transponders
- Frequencies
- Tools

All other practical information remains directly inside each vehicle record.
