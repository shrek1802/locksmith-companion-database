# Ford Legacy Passenger Security Split Audit

**Version:** 1.0  
**Status:** In progress  
**Last updated:** 2026-07-14  
**Scope:** UK-market Escort, Scorpio, Probe, Cougar, Ka, StreetKa, Ka+, Fusion and classic Puma records

## Purpose

This document records provisional record boundaries for lower-volume and legacy Ford passenger models before technical key data is approved.

A facelift is not treated as a security split unless locksmith-relevant evidence proves a change.

## Escort

### Confirmed boundary

- The current inventory covers the late Escort era relevant to transponder/PATS work.
- Escort car production ended in 2000; van derivatives continued later.

### Provisional record plan

| Proposed record | Status | Reason |
|---|---|---|

| Escort Van 1995–2002 keyed | verification_required | Separate badge/body use; verify whether security data matches passenger Escort. |

## Ka and StreetKa

### Confirmed generation boundaries

- First-generation Ka: 1996–2008 on Ford/Fiesta-derived architecture.
- StreetKa: separate convertible derivative sold in the early 2000s.
- Second-generation Ka: 2008–2016, produced with Fiat and based on Fiat 312/Panda architecture.

### Provisional record plan

| Proposed record | Status | Reason |
|---|---|---|
| Ka 1996–2008 keyed | verification_required | Ford-derived first generation; verify any transponder/security breakpoint. |
| StreetKa 2003–2005 keyed | verification_required | Separate badge/body derivative; verify whether security data matches first-generation Ka. |
| Ka 2008–2016 keyed | shared_platform | Fiat 312 / Fiat CODE architecture; keep under Ford badge. |

## Ka+

### Confirmed boundary

- It is unrelated to the Fiat-based second-generation Ka and is closer to Ford's global small-car/Fiesta family.
- The 2018 facelift remains inside the same generation unless security evidence proves a change.

| Proposed record | Status | Reason |

## Evidence Quality

The badge and generation boundaries above are suitable for inventory correction. They do not approve:

- blade profile;
- transponder;
- frequency;
- Add Key or All Keys Lost support;
- tool compatibility;
- module location;
- online/security access requirements.

## Master-File Actions

1. Split Escort car and Escort Van records.
2. Keep Probe as a Mazda-derived shared-platform record.
3. Keep first-generation Ka separate from Fiat-based Ka 2008–2016.
4. Keep StreetKa as its own badge record.
5. Keep Ka+ 2016–2020 separate from both earlier Ka generations.
6. Keep original Puma 1997–2002 separate from Puma 2019–present.
7. Correct Maverick second generation to 2001–2007 rather than 2001–2005 where UK evidence supports continued sale.
